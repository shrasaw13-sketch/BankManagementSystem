# Bank Management System

A Core Java console-based application developed using Object-Oriented Programming concepts.

## Features
- Create Bank Account
- Deposit Money
- Withdraw Money
- Display Account Details

## Technologies Used
- Java
- Eclipse IDE
- GitHub

## OOP Concepts Used
- Encapsulation
- Inheritance
- Abstraction
- Polymorphism
- Method Overriding
